Nevzorov¶

The following example presents the data collected by the Nevzorov probe on Polar 5 during the MOSAiC-ACA campaign.

If you have questions or if you would like to use the data for a publication, please don’t hesitate to get in contact with the dataset authors as stated in the dataset attributes contact or author.

Data access¶

To load the data we first load the (AC)³airborne meta data catalogue. More information on the catalog can be found here.

import ac3airborne

cat = ac3airborne.get_intake_catalog()
list(cat.P5.NEVZOROV)

ds_nevzorov = cat['P5']['NEVZOROV']['MOSAiC-ACA_polar5_RF11'].to_dask()

The dataset includes liquid water content (lwc) and total water content.

Load Polar 5 flight phase information¶

Polar 5 flights are divided into segments to easily access start and end times of flight patterns. For more information have a look at the respective github repository.

At first we want to load the flight segments of (AC)³airborne

meta = ac3airborne.get_flight_segments()

For now we are not doing more then plotting a complete flight.

Plot¶

%matplotlib inline
import matplotlib.pyplot as plt
import matplotlib.dates as mdates
plt.style.use("../mplstyle/book")

fig, ax = plt.subplots(1, 1)

ax.plot(ds_nevzorov.time, ds_nevzorov.lwc, color='k', label='LWC')
ax.plot(ds_nevzorov.time, ds_nevzorov.twc, color='b', label='TWC')

#ax.set_ylim(0, 4)
ax.set_ylabel('water content[kg/m^3]')

ax.legend(frameon=False, loc='upper right')

ax.xaxis.set_major_formatter(mdates.DateFormatter('%H:%M'))

plt.show()
